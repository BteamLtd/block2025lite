# F6R1

## Step 1
请使用```函数```拼图块，完成让小机器人发出货物的任务。你需要完成三次不同的发货任务，分别是发出``5``个、``8``个、``4``个。每次放置好货物，要让小机器人``发出货物``。

```ghost
    function 发出货物 () {
        agent.move(RIGHT, 4)
    }
    function 放置红色羊毛货物 (数量: number) {
        agent.move(RIGHT, 1)
        agent.setItem(RED_WOOL, 数量, 1)
        for (let index = 0; index < 数量; index++) {
            agent.place(FORWARD)
            agent.move(UP, 1)
        }
        agent.move(DOWN, 数量)
    }
    function 归位 () {
        agent.move(LEFT, 4)
    }
    归位()
    放置红色羊毛货物(1)
    发出货物()

```
