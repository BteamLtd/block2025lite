# F8R1

## Step 1
当玩家靠近一个位置时，将这个玩家传送到另一个位置。你可以先执行已经存在的拼图，然后踩上绿色方块上体验一下效果。最终要求玩家靠近迷宫内的绿色方块时，传送到迷宫的出口外的金色方块上。你需要注意各个位置的坐标，你的位置位于屏幕的左上角。
 

```
loops.forever(function () {
    mobs.teleportToPosition(
    mobs.near(
    mobs.target(ALL_PLAYERS),
    world(-41, -54, 95),
    2
    ),
    world(-34, -54, 95)
    )
})
